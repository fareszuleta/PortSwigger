# PortSwigger

A repository dedicated to PortSwigger Web Security Academy challenges and learning materials.

## Overview

This repository contains solutions, notes, and implementations for PortSwigger Web Security Academy labs and challenges. It's a resource for learning web application security through hands-on practice.

## About PortSwigger

PortSwigger Web Security Academy is a comprehensive free online training program that covers:
- Web Security Fundamentals
- Advanced Web Security Concepts
- Practical labs and challenges

## Features

- Lab solutions and walkthroughs
- Security vulnerability explanations
- Exploitation techniques and methods
- Best practices for secure coding
- Burp Suite techniques and usage

## Getting Started

### Prerequisites

- Burp Suite (Community or Professional edition)
- Web browser
- Basic networking knowledge

### Installation

```bash
git clone https://github.com/fareszuleta/PortSwigger.git
cd PortSwigger
```

### Usage

Each directory contains solutions and notes for specific lab topics:
1. Navigate to the relevant security topic folder
2. Review the documentation and lab solutions
3. Follow along with the explanations
4. Practice with the PortSwigger Academy labs

## Topics Covered

- SQL Injection
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Authentication & Session Management
- API Security
- Server-Side Request Forgery (SSRF)
- XML External Entity (XXE) Injection
- And more...

## Tools Used

- **Burp Suite** - Web application penetration testing
- **Browser DevTools** - Analysis and debugging
- **Command-line utilities** - Exploitation and testing
